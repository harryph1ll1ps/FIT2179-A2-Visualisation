# Major nuclear power station accidents and incidents since 1952

- **File:** `nuclear-incidents.csv`
- **Powers:** INES severity timeline — *"Every Major Nuclear Incident Since 1952"*
- **Source:** [GZERO Media — *Forty years since Chernobyl*](https://www.gzeromedia.com/graphic-truth/forty-years-since-chernobyl-is-nuclear-energy-more-essential-than-ever), using the IAEA International Nuclear Event Scale (INES).
- **Coverage:** Selected major events, 1952–2011.

## Columns
| Column | Description |
| --- | --- |
| `year` | Year the incident occurred. |
| `severity` | INES rating (4–7): 4 = accident with local consequences, 5 = accident with wider consequences, 6 = serious accident, 7 = major accident. |
| `title` | Name of the incident or plant (e.g. Chernobyl, Fukushima). |
| `country` | Country where the incident occurred. |
