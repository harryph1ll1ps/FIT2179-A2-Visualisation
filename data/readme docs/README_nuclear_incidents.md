# Nuclear Power Station Accidents and Incidents Since 1952

**Corresponding file:** `nuclear_incidents.csv`

**Source:** [GZERO Media — Forty years since Chernobyl: Is nuclear energy more essential than ever?](https://www.gzeromedia.com/graphic-truth/forty-years-since-chernobyl-is-nuclear-energy-more-essential-than-ever)

### Column Descriptions
| Column | Description |
| --- | --- |
| `year` | Year the incident occurred. |
| `severity` | INES severity rating (4–7), where 4 = accident with local consequences, 5 = accident with wider consequences, 6 = serious accident, 7 = major accident. |
| `title` | Name of the incident or plant. |
| `country` | Country where the incident occurred. |
